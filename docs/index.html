📱 Setup Guide — One Time Only
Works for any public or private channel you follow — ET Markets, CNBC, Bloomberg Quint, etc. No need to own the channel.

Step 1 — Get API credentials
Go to my.telegram.org/apps → log in → create an app → copy API ID and API Hash.

Step 2 — Generate a session string
On your PC/laptop, run once:
pip install telethon
python scripts/generate_telegram_session.py It will ask for your phone number and OTP, then print a long session string. Copy it.

Step 3 — Add GitHub Secrets
Go to your GitHub repo → Settings → Secrets → Actions → add these 3 secrets:
TELEGRAM_API_ID   TELEGRAM_API_HASH   TELEGRAM_SESSION

Step 4 — Add channels below + toggle ON in News Sources
The report will include last 24h messages tagged [TELEGRAM — @channel].
